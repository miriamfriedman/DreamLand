# DreamLand
Full-stack web application built with Angular and C#


DreamLand is a full-stack toy store application.

## Technologies

### Backend

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* C#

### Frontend

* Angular
* TypeScript
* HTML
* CSS

## Features

* Browse products by categories
* Customer registration and login
* Shopping cart management
* Checkout process
* Product search

## Project Structure

```
DreamLand
│
├── DreamLand-backend
│   ├── WebApi
│   ├── Bll_Services
│   ├── Dal_Repository
│   └── Dto_Common_Enteties
│
├── DreamLand-frontend
│   ├── src
│   ├── public
│   └── services
```

## Installation

### Backend

```bash
cd DreamLand-backend
```

Open `DreamLand.sln` with Visual Studio and run the WebApi project.

### Frontend

```bash
cd DreamLand-frontend
npm install
ng serve
```

The application will run on:

```
http://localhost:4200
```
